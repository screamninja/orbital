    <h1 align="center">Тестовое задание</h1>
    <br>
</p>

Необходимо разработать простенькое приложение на базе фреймворка Yii или Yii2 (http://www.yiiframework.com).
Приложение содержит игровую часть и  форму авторизации, содержащую только поле для ввода имени и кнопку ОК. После регистрации пользователь видит квадратное поле 600x600, на котором находится подписанный введенным именем квадратик. Квадратик можно передвигать мышкой. Одновременно может войти несколько пользователей,каждый может управлять только своим квадратиком и видеть перемещение чужих квадратиков в реальном времени (допускается задержка). Имеется ввиду, что можно открыть разные браузеры или режим инкогнито и ввести новый логин чтобы авторизоваться как другой пользователь. Есть кнопка "выход".
